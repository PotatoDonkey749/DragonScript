Lesson 1: Variables

Learn how to assign variables.

Variables are like containers that hold different types of things.

In our programming language, we have different types of variables that we can use:
int: This is a variable that holds whole numbers, like 1, 2, 3, and so on.
str: This is a variable that holds text, like "hello" or "world".
rnd: This is a variable that holds a random number between two other numbers.
ini: This is a variable that holds a whole number that the user types in.
ins: This is a variable that holds text that the user types in.
str/: This is a variable that holds text that represents a whole number.

Lesson 2: Arithmetic and string operations.

Learn how to use arithmetic and string operations.

We can also do math with variables!

Here are the operations we can use to manipulate variables:
inc: This adds 1 to a variable.
dec: This subtracts 1 from a variable.
add: This adds two variables together.
sub: This subtracts one variable from another.
mul: This multiplies two variables together.
div: This divides one variable by another.
con: This combines two pieces of text together

Lesson 3: Input/Output.

Learn how to print variables to the screen.

We can also talk to the computer and ask it to do things for us like to print something to the screen.

Here are the operations to do that:
prt: This prints the value of a variable on the screen.
pra: This prints the ASCII value of a variable on the screen.
prs: This prints the text value of a variable on the screen.

Lesson 4: Conditionals.

Learn how to use jump statements to control the program.

Sometimes we want the computer to do different things depending on what is in a variable.

Here are some commands that help us do that:
lbl: This creates a label that we can jump to later.
ret: This returns to the line after the label was called from.
jmp: This jumps to a different part of the program.
jz: This jumps to a different part of the program if a variable is equal to zero.
jgz: This jumps to a different part of the program if a variable is greater than zero.
jlz: This jumps to a different part of the program if a variable is less than zero.
je: This jumps to a different part of the program if two variables are equal.
jg: This jumps to a different part of the program if one variable is greater than another.
jl: This jumps to a different part of the program if one variable is less than another.
jnz: This jumps to a different part of the program if a variable is not equal to zero.
jne: This jumps to a different part of the program if two variables are not equal.
cmp: This jumps to a different part of the program if two strings are the same.

Lesson 5: Halting.

Learn how to stop the program based on conditionals.

When we reach the end of our program we somehow have to stop it.

Here are some commands to help us do that:
hlt: This stops the program. hz: This stops the program if a variable is equal to zero. hbz: This stops the program if a variable is greater than zero. hlz: This stops the program if a variable is less than zero. hb: This stops the program if one variable is greater than another. hl: This stops the program if one variable is less than another. hne: This stops the program if two variables are not equal. he: This stops the program if two variables are equal.

Lesson 6: Other.

Here are some other commands you can use:

These can be useful but it is not neccesary to know.

not: this toggles a variable between 0 and 1.
slp: this stops the program for a set amount of milliseconds.
push: this pushes a variable to the stack so we can use it later.
pop: this retreives the top value of the stack into a variable.
imp: this imports your code from an other file into this program.
del: this deletes a variable.
;: this declares a comment that you can add notes to your code with.
cmd: this executes a terminal command.
pxi: this draws a pixel on the terminal using integer arguements.
pxv: this draws a pixel on the terminal using variable arguements.
cls: this clears the screen.
